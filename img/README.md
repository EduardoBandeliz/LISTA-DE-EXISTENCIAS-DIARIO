# Imagenes de productos

Guarda aqui las imagenes optimizadas de celulares.

Formato recomendado:

```text
img/CODIGO.webp
```

Ejemplo:

```text
img/848958043799.webp
```

Luego agrega la ruta en `product-images.json`:

```json
{
  "848958043799": "img/848958043799.webp"
}
```

Usa imagenes `.webp` comprimidas para que Netlify cargue rapido.
